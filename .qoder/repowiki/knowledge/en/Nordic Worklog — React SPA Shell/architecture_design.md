Vite + React 19 SPA bootstrapped from `src/main.jsx`, which mounts `<App />` inside a `<ThemeProvider>` (global theme via `src/context/ThemeContext.jsx`). `src/App.jsx` owns all routing as in-process tab state (`activeTab`) and switches between four page components: Home, Entradas, Projetos, Configuracoes. Each page lives under `src/pages/<Name>/` and may ship its own `components/` subfolder for page-scoped UI pieces (e.g. `Projetos/components/ProjectItem.jsx`, `Configuracoes/components/ParametrosBox.jsx`). Shared chrome is factored into `src/components/Layout/Layout.jsx` and `src/components/BottomNav/BottomNav.jsx`. Assets are static imports under `src/assets/`. There is no client-side router; navigation is purely local state passed down through Layout → BottomNav.
